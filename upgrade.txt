Дириктива @extend делат. К примеру имеет клас containter в котором описаны основные стили
так же к етому елементу привязан еще один клас containter--position которы добавляет функционал разсположение даного
блока. Можно в HTML не прописывать два класса а просто указать один containter--position а в 
препроцесоре описать свойства для двух класов и через директиву применить их.

  <div class="container--position">

  .container {
    width: 1200px;
  }

  .container--position {
    @extend .container;
    display: flex;
    align-items: center;
}
